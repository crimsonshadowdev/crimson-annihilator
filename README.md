# 🩸 CRIMSON ANNIHILATOR

Bot WhatsApp autonome avec QR code, pairing code, et commandes dévastatrices.

## Utilisation
1. `npm install`
2. `npm start`
3. Scanne le QR code affiché dans le terminal (WhatsApp > Appareils liés)
4. Si le QR ne fonctionne pas, attends 10 secondes, entre ton numéro et saisis le code d'appairage
5. Utilise les commandes ci-dessous

## Commandes
- `!help` – Menu avec l'image CRIMSON
- `!invite` – Liens chaîne et groupe WhatsApp
- `!crash [numéro]` – 20 crashes visuels
- `!purge` – Expulse tous les membres (admin)
- `!kick @membre` – Expulse un membre (admin)
- `!warn @membre` – Avertit un membre
- `!tagall` – Mentionne tout le monde
- `!appear` – Apparaît comme une chaîne CRIMSON
- `!destroy [numéro]` – Bugs destructeurs
